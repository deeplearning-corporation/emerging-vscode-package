# Emerging Language for VS Code

Emerging 是一种简单易学的编程语言，专为编程初学者设计。

## 特性

- 🎨 **语法高亮** - 清晰的颜色标识
- 🚀 **一键运行** - 直接运行 Emerging 代码
- 💡 **智能补全** - 代码自动完成
- 📝 **代码片段** - 快速插入常用代码结构
- 🔍 **悬停提示** - 鼠标悬停显示帮助信息

## 快速开始

### 1. 创建文件
创建扩展名为 `.emg` 的文件

### 2. 编写代码
```emerging
// 第一个程序
let name = "World"
print "Hello, " + name + "!"

// 算术运算
let a = 10
let b = 20
print a + b

// 函数
function greet(person) {
    print "Hello, " + person
}
greet("Emerging")
